# Changelog

## 0.1.0

Initial release.

- Bar, line, and pie chart types via Chart.js
- Inline spreadsheet-style data editing
- CSV file upload and paste import with validation
- Configurable color palette and color generator
- Customizable icons, class names, and Chart.js options
- Slash menu integration via `insertChart`
- CSS custom properties for theming (`--bn-chart-*`)
- Dropdown positioning via `@floating-ui/react`
